# Basic Sentiment Analysis Tool

This is a simple command-line tool that analyzes the sentiment of a sentence and classifies it as positive, negative, or neutral.

## Purpose

The goal of this project was to explore the basics of Natural Language Processing (NLP) and sentiment analysis. It demonstrates the use of external libraries and the application of machine learning concepts to real-world text data.

## How to Run

1.  **Clone the repository:**
    ```
    git clone https://github.com/your-username/sentiment-analyzer.git
    ```
2.  **Install the necessary dependencies:**
    ```
    pip install -r requirements.txt
    ```
3.  **Download the necessary TextBlob corpora:**
    ```
    python -m textblob.download_corpora
    ```
4.  **Run the script:**
    ```
    python sentiment_analyzer.py
    ```
